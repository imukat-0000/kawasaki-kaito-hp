# 河﨑海斗 公式サイト — Claude Code運用ルール

このリポジトリは河﨑海斗（金工作家・鋳金）の公式サイト。`index.html` 1ファイル完結（依存なし・ビルド不要）。
`main` にpushすると GitHub Pages（https://imukat-0000.github.io/kawasaki-kaito-hp/）に自動反映される。

## デザイン・コンセプト（変更時は踏襲すること）

- コンセプト：「KOGEI REALISM（工芸リアリズム）＝存在の条件をつくるリアリズム」
- 表記は「河﨑海斗」（﨑）で統一。「河崎」は使わない
- 文章は**一人称を使わない**（本人が語るのではなく、正典テキストの語り口を踏襲）
- 作品紹介はモチーフの説明から入らない。「存在の条件」の側から言葉を組み立てる
- アクセントカラーは緋銅赤 `#9C3B22`（ダークテーマでは `#CE6A4A`）。トーンを変える場合もこの系統から外さない
- 和文見出しは明朝体（Noto Serif JPサブセットを `fonts/` に同梱）、英字見出しはDidot系

## 技術的な注意点

- 1ファイルHTML。フレームワークやビルドツールは使わない方針
- 金魚のオープニングアニメーション・カーソル追従・「餌をあげて育てる」隠しゲームはCanvas上のvanilla JSで実装（`<script>` 内）
- 画像を追加する場合は `img/` に置き、大きい写真は書き出し時に軽量化する（サイト全体で数MB以内を目安に）
- 変更後は実際にブラウザで開いて動作確認してから commit・push すること

## 作業ルール

- 変更したら `git add` → `git commit` → `git push` まで行う（ユーザーから明示的な指示がなくても、編集が一区切りついたら「commitしてpushしましょうか」と確認してよい）
- 大きな構成変更（ナビゲーション構造・使用データベース等）は、正本データのある河﨑・タクミ間のObsidian vaultとズレる可能性があるため、変更前に一声かける
